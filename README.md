# basic-js
a basic JS toolkit I use for teaching JS

# Things you'll need
- git
sudo apt-get install git
- node.js/npm
curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
sudo apt-get install -y nodejs

# Getting Started
- Git Clone the project
- run npm install
- run npm start
- start editing src/index.js
